# Resources Directory

## Icon File

Please add an application icon file named `app.ico` to this directory.

The icon will be used for:
- The application executable
- The installer
- Desktop shortcuts

Recommended icon specifications:
- Size: 256x256 pixels (high DPI)
- Format: ICO with multiple sizes (16x16, 32x32, 48x48, 64x64, 128x128, 256x256)
- Color depth: 32-bit with alpha channel

You can create an ICO file from PNG using online tools or image editing software like:
- GIMP
- Adobe Photoshop
- Online converters (e.g., convertico.com)
